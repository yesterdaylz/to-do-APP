# 柠檬Todo

### 简介

一个简易的待办app,仅供学习参考。

可用于设置待办，计时统计等功能（详见已实现功能或文件夹里的UpdateLog）

嗯？你问我为什么选待办？？？只不过因为本人用过的待办软件比较多（大概有六七种吧）~~其实只是觉得比较简单而已~~

app名由来？随便取的😊.

### 已实现功能

- 底部导航栏
- 抽屉
- 待办的增删查改
- 待办通知
- 登录注册(本地存储）
- 用户信息存储（本地存储)
- 多用户支持
- 个人信息页
- 番茄钟功能（正倒计＋番茄）
- 播放背景音乐
- 最近七日柱状图统计
- 随机名言（一言接口）
- 深色模式
- 中英切换
